# 🎬 Cartelera CRUD

Proyecto de fin de curso — CRUD completo de películas en cartelera.

## Tecnologías

- **Frontend:** Angular 17 + SCSS + SweetAlert2
- **Backend:** Node.js + Express + Sequelize
- **Base de datos:** SQLite
- **Autenticación:** JWT

## Funcionalidades

- ✅ Login con JWT
- ✅ Listar películas con filtros por género y búsqueda
- ✅ Crear película con validaciones
- ✅ Editar película
- ✅ Ver detalle de película
- ✅ Eliminar con confirmación
- ✅ Mensajes de éxito y error personalizados

## Instalación local

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend
```bash
cd frontend
npm install
ng serve
```
